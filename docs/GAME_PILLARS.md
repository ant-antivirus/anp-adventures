# ANP Adventures Game Pillars

Core line:

"โลกแห่งการผจญภัยที่เด็กและครอบครัวสร้างความทรงจำร่วมกัน โดยคุณค่าของผู้เล่นไม่ได้วัดจากการเติมเงิน"

ANP Adventures is a family-friendly Roblox adventure world where memories matter more than power.

## Design Pillars

1. Adventure Before Grinding
2. Social Before Competition
3. Memories Before Numbers
4. Fair Before Profit
5. Build for Growth

## Decision Question

"Does this feature create adventure, relationships, or memories?"

## Development Implications

- Episodes should prioritize discovery, story, character moments, and secrets.
- Social features should support helping, sharing, showing achievements, and playing together.
- Progression should not become repetitive grinding.
- Monetization must not sell power.
- EP1 only needs to prove that the world is fun enough to revisit.
